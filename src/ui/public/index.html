Browser Integration

This application uses your existing browser session to download invoices automatically. Please ensure:

  1. You are already logged into your accounts (Amazon, Google, etc.) in Chrome or Edge
  2. Close all Chrome/Edge windows before starting to avoid port conflicts
  3. The application will launch and control your browser automatically
  4. If the system browser doesn't work, a Playwright-managed browser will be used as fallback

Note: If you encounter "Connection Refused" errors, try manually closing all browser windows and restarting the application.
